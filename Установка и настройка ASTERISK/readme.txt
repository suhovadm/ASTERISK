Гайды по установке и настройке готовой сборки Sangoma Linux 7 + Asterisk 18.16 + freePBX 16.0.33.
Несколько штук - отдельный гайд по установке, отдельный гайд по настройке.

В данном случае, рассматривается вариант установки в виртуальной машине Virtual Box 6.1.
Таким образом, можно будет установить данную линуксовую сборку IP телефонии на любой виндовый сервер, или компьютер.
